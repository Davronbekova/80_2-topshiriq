## React DApp qismi

- `src/App.jsx`
- `src/main.jsx`
- `src/styles.css`

## Qanday ishga tushirish

1. `npm install`
2. `npm run dev`
3. Brauzerda `http://localhost:5173` ni oching

## Muhim

- `src/App.jsx` ichidagi `CONTRACT_ADDRESS` qiymatini o‘z testnet kontraktingiz adresiga o‘zgartiring.
- Sepolia testnet RPC: `https://rpc.sepolia.org`
