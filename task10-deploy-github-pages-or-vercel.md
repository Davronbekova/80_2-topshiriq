# Task 10 - Deploy GitHub Pages or Vercel

## GitHub Pages
1. GitHub repositoriyangizga push qiling.
2. `Settings > Pages` bo‘limidan `gh-pages` yoki `main/master` branch-ni tanlang.
3. Build fayllari `dist` papkasidan chiqsin.
4. Agar kerak bo‘lsa `package.json` ga `homepage` maydonini qo‘shing.

## Vercel
1. GitHub repozitoriyangizni Vercelga ulang.
2. `Framework preset` sifatida `Vite` ni tanlang.
3. Build komandasi: `npm run build`.
4. Output papkasi: `dist`.
5. Deploy tugmasini bosib saytga ulaning.
