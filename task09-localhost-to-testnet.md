# Task 09 - Localhost to Testnet

1. Lokal localhost serverda React DApp ni ishga tushiring: `npm install` va `npm run dev`.
2. Smart-kontrakt testnetga (Sepolia) ulanish uchun `src/App.jsx` ichidagi RPC URLni `https://rpc.sepolia.org` ga o‘zgartiring.
3. Kontrakt adresini ham Sepolia testnet adresiga moslang.
4. Har bir wallet ulanishi uchun MetaMask yoki WalletConnect testnet sozlamalarini tekshiring.
5. Endi DApp browserda `http://localhost:5173` orqali ishlaydi, lekin kontrakt Sepolia tarmog‘ida.
